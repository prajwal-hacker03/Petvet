PetCare Guide 🐾

PetCare Guide is a responsive pet rescue and pet care website built using HTML, CSS, and JavaScript. The website helps people care for abandoned pets by providing complete guidance about feeding, vaccinations, grooming, rescue tips, and nearby veterinary support.

Features ✨
Pet categories like Dogs, Cats, Birds, Rabbits, etc.
Age-wise pet care guidance
Feeding and nutrition information
Vaccination schedules
Rescue and emergency care tips
Nearby vet section
Adoption section
Responsive and modern UI
Dark mode support
Smooth animations and interactive design
Technologies Used 💻
HTML
CSS
JavaScript
Project Goal 🎯

The main goal of this project is to help first-time pet adopters and animal rescuers understand how to properly care for pets from rescue to adulthood.
